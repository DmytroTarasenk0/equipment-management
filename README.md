# Equipment Lifecycle Management

A responsive web application mockup designed to track and manage the lifecycle and maintenance statuses of medical equipment. Created as a laboratory project for the "WEB-oriented technologies" course.

## Features

- **Responsive Layout:** Adapts seamlessly to desktop, tablet, and mobile screens using CSS Flexbox and Grid.
- **Mobile Navigation:** Interactive burger menu for smaller screens.
- **Status Indication:** Visual badges indicating the current state of equipment (Active, Warning, Maintenance, Decommissioned).
- **Hover Animations:** Smooth CSS transitions for better user experience (UX).

## Technologies Used

- HTML5 (Semantic markup)
- CSS3 (Custom properties, Media queries, Flex/Grid)
- Vanilla JavaScript (DOM manipulation)

## How to Run Locally

This project does not require any build tools or external dependencies.

1. Clone this repository to your local machine:
   `git clone https://github.com/DmytroTarasenk0/equipment-management.git`
2. Navigate to the project folder.
3. Open the `index.html` file in any modern web browser.
   _Alternatively, you can use the "Live Server" extension in Visual Studio Code for a better development experience._
